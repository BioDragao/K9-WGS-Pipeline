# K9-WGS-Pipeline
[![Travis status][travis-badge]][travis-link]

Nextflow pipeline for standardised variant calls on canine genomes

## Usage

```bash
$ nextflow run main.nf
```

## Run on test data

```bash
$ tar xjvf test-data/test-data-tiny.tar.bz2 -C test-data
$ nextflow run main.nf
```
